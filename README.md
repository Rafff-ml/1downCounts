# 1downCounts
1downCounts
1downCounts is a web-based scoring application designed to manage street cricket matches. It replaces manual counting by providing a digital interface to track runs, wickets, and player statistics in real-time.
Functionality
Match Setup
Configure team names and set the number of overs per innings.
Input and manage player rosters for both teams.
Simulate a coin toss to determine which team bats or bowls first.
Scoring System
Track runs, wickets, and ball count per over.
Record extras including Wides and No-balls.
Specify dismissal types (Bowled, Catch, LBW, Run Out).
Calculate the target score and remaining balls required in the second innings.
Player Management
Track individual batsman scores and balls faced.
Track bowler figures including overs bowled, runs conceded, and wickets taken.
Automatically rotate strike between batsmen on single runs and over completion.
Data Persistence
The application automatically saves the match state to the browser's local storage after every ball.
Users can close the browser and resume the match from the exact point of interruption.
Sharing
Generates a formatted text summary of the current match score, batter status, and bowler statistics for clipboard copying.
